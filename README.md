# 💼 SmartHire AI

SmartHire AI is an AI-powered Resume Screening and Job Recommendation System built using Machine Learning, Natural Language Processing (NLP), and Streamlit.

The application analyzes resumes, predicts the most suitable career category, recommends relevant jobs based on resume content, and calculates how well a resume matches a given job description.

---

## ✨ Features

- 📄 Resume Parsing (PDF & DOCX)
- 🤖 Resume Classification
- 💼 Job Recommendation
- 📊 Resume Fit Prediction
- 📈 Job Clustering (Exploratory)
- 🌐 Interactive Streamlit Web Application

---

## 🛠 Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- TF-IDF
- Cosine Similarity
- Joblib

---

## 🚀 Running the Project

Clone the repository

```bash
git clone https://github.com/navya7003/SmartHire_AI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app/streamlit_app.py
```
